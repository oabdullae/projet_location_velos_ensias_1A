# projet_location_velos_ensias_1A
Projet de gestion de location de vélos en C, réalisé dans le cadre du module de langage C à l’ENSIAS (1ʳᵉ année).  L’application permet de gérer un centre de location de vélos à travers un menu textuel interactif. Elle implémente toutes les fonctionnalités exigées: opérations CRUD, tri, recherche, statistiques, l’import/export des données.
